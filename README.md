# TalentKeepers

## Getting Started

```
    Install [NodeJS](https://nodejs.org/en) lts version(18.16.0)
```

### 1. Clone this template:

 clone:
 ```bash
 git clone git@github.com:gyanminda/TalentKeepers.git
 ```

### 2. Installing dependencies

```bash
npm install
```
### 3. run the dev server

```bash
npm run dev
```
